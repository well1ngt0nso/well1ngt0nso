## 📌 O que você vai encontrar aqui?

Este repositório é um compilado de projetos, estudos e soluções que unem hardware, software e mecânica. Os principais focos são:

* **⚡ Microcontroladores e Sistemas Embarcados:** Projetos práticos utilizando ESP32, ESP8266, Arduino, Raspberry Pi e modelos semelhantes (comunicação serial, servidores web, IoT).
* **🔌 Eletrônica e Projetos de Hardware:** Desenvolvimentos de circuitos, análise de sistemas e aplicação de conceitos de engenharia.
* **🧠 Inteligência Artificial e Machine Learning:** Modelos de Redes Neurais (das simples às mais complexas) aplicadas a problemas reais e controle de sistemas, utilizando Keras e TensorFlow.
* **⚙️ Modelagem 3D e CAD:** Projetos de design mecânico, plantas, modelagem no FreeCAD e arquivos otimizados para impressão 3D.
* **💻 Programação Aplicada:** Scripts estruturados em C++ e Python para automação, cálculos e geração de dados.
